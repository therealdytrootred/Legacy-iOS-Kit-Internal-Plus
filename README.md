# Legacy iOS Kit Internal Plus (Unofficial Fork of Legacy iOS Kit by LukeZGD and Kaidencc)

- formerly iOS-OTA-Downgrader  
- all-in-one tool for restoring, downgrading, jailbreaking, SHSH blobs, and more  
- supports macOS and Linux

## Features

- Support for most 32-bit iOS devices and some 64-bit devices  
- Restore to signed OTA versions (e.g. iOS 8.4.1, 6.1.3)  
- Restore to unsigned versions using SHSH blobs  
- Jailbreak support for legacy iOS versions  
- SSH Ramdisk support  
- SHSH blob saving (onboard + external)  
- IPA sideloading  
- App and data management tools  
- Various utilities (NVRAM, pairing, diagnostics, etc.)  


## Supported Platforms

- macOS 10.11 and newer  
- Linux (Ubuntu, Debian, Fedora, Arch, etc.)  
- Architectures: x86_64, arm64  


## Usage

Run the script with


./restore.sh
